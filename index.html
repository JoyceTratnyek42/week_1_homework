4: Describe the relationship between HTML, CSS, and Javascript. Which one(s) are required to create a webpage?

You can create a basic website with just html or html and css, but all three are used for most modern, profesisonal websites. HTML mostly lays out the skeleton of a website's content through text and simple styling. CSS adds more styling and personalization, and JS adds the website's dynamics and interactivity, allowing it to respond to input from users, so it is no longer static until manually updated.

14: Go back to the CSS gradient website. Drag the color slider around and see how the generated CSS changes. Move the position handles for each color and do the same. Describe how the generated CSS changes in response to these changes. Can you figure out how a gradient is defined in CSS?

The three parts of the 2nd "background" parameter define the numeric color values of each point in red, green, blue, and transparent parameters. The percentage at the end of each part refers to when in the overall gradient that specific color point hits, so my three are set to 0%, 50% and 100%. Changing these would control how much space each color point takes up in the overall gradient.
